## 2 - Chip8 Features

Here are some features:
- Monochrome 64x32 pixels display
- 4KB of Memory
- 512B reserved
- 3.5KB of RAM shared with the game program
- 16 x 8 bit registers
- 16 key keypad input
- delay timer
- sound timer
- 36 instructions
